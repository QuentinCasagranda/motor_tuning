# Motor Tuning documentation

This repository contains material for setting and tuning motor's control command on a PLC running under TwinCat
(Beckhoff PLCs).

Goal is to share procedure and key files for setting up TwinCat 3 to drive motors.

So far it has one documentation which can be accessed [On the readthedoc website](https://motor-tuning.readthedocs.io/en/latest/dc_7342.html)
